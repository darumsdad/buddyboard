BuddyBoard

Real-Time Pool Safety Tracking for Camp Regesh
Project Overview  ·  Prepared June 2026  ·  Confidential

What Is BuddyBoard?

BuddyBoard is a web-based safety system that gives camp pool counselors a live, shared view of every camper currently in the water. When two kids enter the pool together, a counselor registers them as a buddy pair. That pair appears instantly on every device at that pool — so all counselors always know exactly who is in the water, in real time.

When kids leave the pool, their pair is removed. When it's time for a buddy call, counselors open a clean, easy-to-read screen and call off every pair by name, checking each one off as they're confirmed. If everyone answers, the pool is safe.

Core purpose: Every counselor at every pool can always see — in real time — exactly which camper pairs are currently in the water, so no child goes unaccounted for.

Key Features

Fast Buddy Check-In

  • Counselor enters two campers to form a pair
  • Look up by camper code (fastest — single field)
  • Or look up by name typeahead (start typing, pick from list)
  • Pair appears on all devices at that pool instantly
  • Prevents the same camper from being in two pairs at once

Live Buddy Board

  • All current pairs shown in a scrollable list
  • Live swimmer count: "12 swimmers · 6 pairs"
  • Updates in real time as counselors add or remove pairs
  • Multiple counselors can add pairs simultaneously
  • Clear alert when disconnected — never silently stale

Buddy Call Screen

  • Full-screen, high-contrast view optimized for outdoor use
  • Large pair count and swimmer count displayed prominently
  • Scrollable list of all active pairs, each with a checkbox
  • Check off each pair as they answer — names strike through to show who's been accounted for
  • Stays live: updates if a pair is added or removed mid-call

Camper Roster Management

  • Bulk import via Excel spreadsheet upload
  • Add, edit, or remove individual campers at any time
  • Each camper has a staff-assigned code for fast check-in
  • Search and filter the full roster by name or code
  • Clear all campers at once when the summer ends

Pool & Session Management

  • Configurable pool list (set up by admins)
  • Pool picker shows a Live badge on pools with an active session
  • Session opens automatically when a counselor enters a pool
  • Multiple counselors can join the same pool's session simultaneously
  • Confirmation required before closing a session with active pairs
  • When a session closes, all connected devices redirect automatically
  • Session history is archived — records are never deleted

User & Account Management

  • Counselors log in with username and password
  • Sessions persist across browser restarts (5-hour idle window)
  • Logged-in username shown on every screen next to the logout button
  • Admins add and remove staff accounts
  • Admins reset passwords directly — no email flow needed
  • Admin panel is separate from the counselor-facing pool screens

Safety Design Decisions

Safety-critical design: Several features are deliberately designed around the consequence of getting them wrong — a child unaccounted for in a pool.

Real-Time Across All Devices

When any counselor at Pool 2 registers or removes a pair, every other device connected to Pool 2 updates within seconds — no page refresh, no manual sync. Multiple counselors can check kids in simultaneously without stepping on each other.

Connection Status Always Visible

If a tablet loses its connection to the system, a prominent banner immediately alerts the counselor: "Disconnected — data may be stale." A board that silently stops updating is more dangerous than one that clearly shows it has a problem.

Remove Requires Server Confirmation

When a counselor removes a buddy pair, the pair does not disappear from the screen until the server confirms the removal. This prevents a pair from vanishing from the board while the children are still in the water due to a network hiccup.

Duplicate Detection

The system prevents the same camper from being registered in two pairs at the same time — even if two counselors attempt to add them simultaneously. The database enforces this rule; it cannot be bypassed by a fast enough click.

Session Close Broadcasts to All Devices

When a counselor closes a pool session, every other device currently viewing that pool is automatically redirected back to the pool list. This ensures no device is left showing a stale, closed session as if it were still active.

Multi-Pool Support

Each pool has its own independent buddy board. Counselors at Pool 1 see only Pool 1's pairs; counselors at Pool 2 see only Pool 2's pairs. Multiple counselors can manage the same pool simultaneously — two or three people checking kids in at once, all seeing each other's updates live.

When a camper switches pools, they are removed from their current pool's board by that counselor, and re-registered as a new pair when they arrive at the next pool. Each pool's board stays independent and clean.

Counselor Quick-Start Guide

Everything a counselor needs to run a safe pool swim from start to finish.

Starting a Swim Session

  1. Open the BuddyBoard URL in your browser and log in with your username and password. Your name will appear in the top corner of every screen so you know you're logged into the right account.

  2. On the Pool List screen, tap your pool. If another counselor already opened a session, you'll see a Live badge on that pool and a confirmation prompt when you enter — tap Join Session to connect to the live board.

  3. The Session Board opens. Any pairs already registered appear immediately. You're live.

Registering a Buddy Pair

  1. In the Add Pair form at the top of the session board, enter the first camper. The fastest method is their camper code — type it in the code field and press Tab or Enter. The camper's name will fill in automatically.

  2. If you don't have the code, type the camper's name in the name field. A list of matches appears — tap the right camper to select them.

  3. Repeat for the second camper, then tap Add Pair. The pair appears on the board immediately for all counselors at this pool.

Tip: The system will warn you if a camper is already in an active pair — this prevents double-counting.

Removing a Pair (Kids Leave the Pool)

  1. Find the pair in the list on the session board.

  2. Tap the × remove button next to their names. The pair disappears from all devices once the server confirms the removal.

Running a Buddy Call

  1. Tap the Buddy Call button on the session board. This opens a full-screen view designed for outdoor, bright-sunlight conditions.

  2. The screen shows the total pair count and swimmer count in large numbers, followed by a list of every active pair.

  3. Call each pair by name. When they answer, tap the checkbox next to their names — it strikes through to show they've been accounted for.

  4. When all pairs are checked off, every camper in the water is accounted for. If a pair doesn't answer, their names remain unchecked — a clear visual cue.

  5. Tap Back to board to return to the session board.

Note: Check-off marks are local to your device and reset when you navigate away — they're a working tool during a call, not a permanent record.

Ending a Swim Session

  1. When swim is over and all kids are out of the water, make sure all pairs have been removed from the board (the swimmer count should read 0).

  2. Tap Close Session in the top-right corner of the session board.

  3. If any pairs are still active, you'll see a warning — confirm you want to close anyway, or go back and remove them first.

  4. Once confirmed, the session closes. All other devices connected to this pool are automatically redirected to the pool list.

Admin Quick-Start Guide

Admin accounts have access to a separate management panel for setting up and maintaining the system.

Accessing the Admin Panel

  1. Log in with an admin account. On the Pool List screen, you'll see an Admin link in the top bar — tap it.

  2. The admin panel has three sections: Users, Campers, and Pools.

Setting Up Counselor Accounts (Users)

  1. Go to Users and tap Add User.

  2. Enter the counselor's first name, last name, username, a temporary password, and their role (User for a regular counselor, Admin for someone who needs management access).

  3. Share the username and temporary password with the counselor. They can log in immediately — there's no email verification step.

  4. To reset a forgotten password, find the user in the list and tap Reset Password.

  5. To remove a staff member's access, tap Delete next to their account.

Loading the Camper Roster

  1. Go to Campers. For a new season, tap Import from Excel and upload your roster spreadsheet. Campers are added in bulk.

  2. To add a single camper, tap Add Camper and fill in their name, bunk, and camper code.

  3. To update a camper's details (e.g., correct a name or change their code), tap Edit next to their row.

  4. At the end of summer, use Clear All Campers to reset the roster before importing next year's list.

Tip: Camper codes are the key to fast check-in at the pool. Make sure every camper has a unique code before the first swim day — short codes (3–4 characters) work best.

Managing Pools

  1. Go to Pools to see the list of configured pools.

  2. Tap Add Pool to create a new pool and give it a name (e.g., "Main Pool", "Lap Pool").

  3. Tap Edit to rename an existing pool. Changes are reflected immediately on the counselor-facing pool list.
